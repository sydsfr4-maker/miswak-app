# 🚀 خطوات بناء APK مسواك عبر GitHub Actions

## شنو راح تحصل؟
ترفع هذا المشروع على GitHub → يبني APK تلقائياً بالسحابة → تحمّله جاهز.
**لا تحتاج كمبيوتر ولا Android Studio ولا أي برنامج.**

---

## الخطوة ١: إنشاء حساب GitHub (مجاني)
١. افتح github.com
٢. اضغط "Sign up"
٣. أدخل إيميل + كلمة مرور + اسم مستخدم
٤. أكّد إيميلك

---

## الخطوة ٢: إنشاء Repository جديد
١. بعد الدخول، اضغط الزر الأخضر **"New"** أو **"+"**
٢. اسم الـ repository: `miswak-app`
٣. اختر **Public**
٤. اضغط **"Create repository"**

---

## الخطوة ٣: رفع الملفات
عندك خيارين:

### الخيار الأسهل (من الموبايل):
١. افتح الـ repository الجديد
٢. اضغط **"uploading an existing file"**
٣. ارفع كل محتويات مجلد `android-project` (package.json, capacitor.config.json, www/, android/, .github/)

### الخيار الأسرع (لو عندك GitHub Desktop):
١. نزّل GitHub Desktop
٢. Clone الـ repository
٣. انسخ ملفات `android-project` داخله
٤. Commit + Push

---

## الخطوة ٤: انتظر البناء التلقائي
١. بعد رفع الملفات، روح لتبويب **"Actions"** بالـ repository
٢. شوف الـ workflow "Build Miswak APK" - راح يبدأ تلقائي
٣. انتظر ١٠-١٥ دقيقة
٤. لو ظهر ✅ أخضر = نجح البناء

---

## الخطوة ٥: تحميل الـ APK
١. اضغط على الـ workflow اللي خلص
٢. اسحب للأسفل لقسم **"Artifacts"**
٣. اضغط **"miswak-debug-apk"** لتحميله
٤. فك الضغط عن الملف المحمّل
٥. ثبّت `app-debug.apk` على تلفونك

---

## ⚠️ ملاحظة مهمة قبل التثبيت
لازم تفعّل "السماح بمصادر غير معروفة" بتلفونك:
- إعدادات → أمان → تثبيت من مصادر غير معروفة → فعّل

---

## شنو بعد كذا؟
- لو اشتغل كل شي زين → Google Login يشتغل بدون مشاكل ✅
- لو تريد APK موقّع (للنشر على Google Play) → نضيف خطوة التوقيع لاحقاً
